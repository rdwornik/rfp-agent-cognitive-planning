This project is a small RFP answer engine for Blue Yonder Cognitive Planning.
It uses:

- A curated **knowledge base** (KB) of historical Tech Presales answers
- A **canonicalization step** to consolidate multiple answers into one
- Gemini **File Search** to retrieve KB content
- A **batch script** that reads RFP CSVs and produces answer CSVs

The goal is to generate pragmatic, SaaS-first, non-overpromising RFP answers
in a consistent style.
